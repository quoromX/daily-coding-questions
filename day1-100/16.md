**Daily Coding Challenge - Day 16: Top Seller**

Struct embedding lets one struct include another struct directly. The embedded struct's fields can be accessed from the outer struct.

Write a function that finds the top active seller.

Create an `Account` struct with:

- `Name`
- `Active`

Create a `Seller` struct that embeds `Account` and also has:

- `Sales`
- `Rating`

The function `topSeller` should accept a slice of `Seller` values.

Each seller's score is:

```go
score = Sales * Rating
```

Return the name of the active seller with the highest score.

Ignore sellers where `Active` is `false`.

If two active sellers have the same score, return the one that appears first.

If there are no active sellers, return an empty string.

**Examples**
`topSeller([]Seller{{Account: Account{Name: "Ada", Active: true}, Sales: 10, Rating: 5}, {Account: Account{Name: "Ben", Active: true}, Sales: 8, Rating: 9}})` returns `"Ben"`
*Explanation:*
Ada has score `50`, and Ben has score `72`.

`topSeller([]Seller{{Account: Account{Name: "Tola", Active: false}, Sales: 20, Rating: 10}, {Account: Account{Name: "Mina", Active: true}, Sales: 3, Rating: 4}})` returns `"Mina"`
*Explanation:*
Tola has the highest score, but is inactive.

**Test Cases**
`topSeller([]Seller{})` = `""`
`topSeller([]Seller{{Account: Account{Name: "Ada", Active: true}, Sales: 10, Rating: 5}, {Account: Account{Name: "Ben", Active: true}, Sales: 8, Rating: 9}})` = `"Ben"`
`topSeller([]Seller{{Account: Account{Name: "Tola", Active: false}, Sales: 20, Rating: 10}, {Account: Account{Name: "Mina", Active: true}, Sales: 3, Rating: 4}})` = `"Mina"`
`topSeller([]Seller{{Account: Account{Name: "A", Active: true}, Sales: 5, Rating: 5}, {Account: Account{Name: "B", Active: true}, Sales: 5, Rating: 5}})` = `"A"`
`topSeller([]Seller{{Account: Account{Name: "Closed", Active: false}, Sales: 100, Rating: 100}})` = `""`

type Account struct {
	Name   string
	Active bool
}

	Account

	Rating int
}

}
