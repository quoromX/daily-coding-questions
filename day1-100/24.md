**Daily Coding Challenge - Day 24: Find All Pattern Occurrences**

This is inspired by ascii-art-color.

Efficient substring search is useful for coloring repeated matches. 

Write a function that returns all starting indices where `pattern` appears in `text`.

Rules:
- Return indices in increasing order.
- Overlapping matches are allowed.
- If `pattern` is empty, return empty slice.
- If no match exists, return empty slice.

**Examples**
`findAllOccurrences("ababa", "aba")` returns `[]int{0, 2}`

`findAllOccurrences("aaaaa", "aa")` returns `[]int{0, 1, 2, 3}`

`findAllOccurrences("hello", "world")` returns `[]int{}`

**Test Cases**
`findAllOccurrences("", "a")` = `[]int{}`
`findAllOccurrences("abc", "")` = `[]int{}`
`findAllOccurrences("abcabcabc", "abc")` = `[]int{0,3,6}`
`findAllOccurrences("mississippi", "issi")` = `[]int{1,4}`
