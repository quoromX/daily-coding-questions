**Daily Coding Challenge - Day 27: Count Islands in a Character Grid**

Flood fill problems show up often when working with grids, maps, or connected regions in ASCII art.

Write a function that counts the number of "islands" in a 2D rune grid, where an island is a group of connected `'#'` characters (connected horizontally or vertically, not diagonally).

The function `countIslands` should accept `grid [][]rune`.

Return the number of distinct islands as an `int`.

Rules:
- Only `'#'` characters count as land; everything else (e.g. `'.'`) is water.
- Cells are connected if they are adjacent horizontally or vertically (not diagonally).
- If the grid is empty, return `0`.

**Examples**
For grid:
```
# # . . #
# . . # #
. . . . .
# # . # .
```
`countIslands` returns `4`

For grid:
```
# # #
# # #
```
returns `1`

`countIslands([][]rune{{'#'}})` = `1`
