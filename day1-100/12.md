**Daily Coding Challenge - Day 12: Closest Point**

A struct groups related fields into one custom type. In Go, structs are useful when one value needs to carry several pieces of data together.

Write a function that finds the point closest to the origin `(0, 0)`.

The function `closestPoint` should accept a slice of `Point` values.

Each point has an `X` and `Y` coordinate.

Use squared distance so you do not need square roots:

```go
distance = X*X + Y*Y
```

Return the closest point.

If two points have the same distance, return the one that appears first.

If the slice is empty, return `Point{}`.

**Examples**
`closestPoint([]Point{{X: 3, Y: 4}, {X: 1, Y: 1}})` returns `Point{X: 1, Y: 1}`
*Explanation:*
The first point has distance `25`, and the second has distance `2`.

`closestPoint([]Point{{X: -2, Y: 0}, {X: 0, Y: 2}})` returns `Point{X: -2, Y: 0}`
*Explanation:*
Both points have distance `4`, so the first one is returned.

**Test Cases**
`closestPoint([]Point{})` = `Point{}`
`closestPoint([]Point{{X: 3, Y: 4}, {X: 1, Y: 1}})` = `Point{X: 1, Y: 1}`
`closestPoint([]Point{{X: -2, Y: 0}, {X: 0, Y: 2}})` = `Point{X: -2, Y: 0}`
`closestPoint([]Point{{X: 5, Y: 5}, {X: -1, Y: -1}, {X: 2, Y: 0}})` = `Point{X: -1, Y: -1}`
`closestPoint([]Point{{X: 0, Y: 0}, {X: 10, Y: 10}})` = `Point{X: 0, Y: 0}`
