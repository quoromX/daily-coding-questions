**Daily Coding Challenge - Day 14: Largest Shape Area**

An interface describes behavior. In Go, a type satisfies an interface when it has the methods the interface requires.

Write a function that finds the largest area from a list of shapes.

Create a `Shape` interface with one method:

- `Area() int`

Then create two structs:

- `Rectangle` with `Width` and `Height`
- `Square` with `Side`

Both structs should have an `Area()` method.

The function `largestArea` should accept a slice of `Shape` values and return the largest area.

If the slice is empty, return `0`.

**Examples**
`largestArea([]Shape{Rectangle{Width: 3, Height: 4}, Square{Side: 5}})` returns `25`
*Explanation:*
The rectangle area is `12`, and the square area is `25`.

`largestArea([]Shape{Square{Side: 2}, Rectangle{Width: 10, Height: 1}})` returns `10`
*Explanation:*
The square area is `4`, and the rectangle area is `10`.

**Test Cases**
`largestArea([]Shape{})` = 0
`largestArea([]Shape{Rectangle{Width: 3, Height: 4}, Square{Side: 5}})` = 25
`largestArea([]Shape{Square{Side: 2}, Rectangle{Width: 10, Height: 1}})` = 10
`largestArea([]Shape{Rectangle{Width: 7, Height: 6}})` = 42
`largestArea([]Shape{Square{Side: 0}, Rectangle{Width: 2, Height: 8}})` = 16

type Shape interface {
	Area() int
}

type Rectangle struct {
	Width  int
	Height int
}

type Square struct {
	Side int
}

}

func (s Square) Area() int {
}

}
