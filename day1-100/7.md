**Daily Coding Challenge - Day 7: Variadic Average Score**

Variadic functions can be called with any number of trailing arguments. For example, fmt.Println is a common variadic function.

Write a function that calculates the average score from any number of scores.

The function should accept one normal boolean parameter, `dropLowest`, followed by a variadic list of integer scores.

If `dropLowest` is `true`, remove only the lowest score before calculating the average. If there are no scores left after dropping the lowest score, return `0`.

If no scores are provided, return `0`.

**Examples**
`averageScore(false, 80, 90, 100)` returns `90`
*Explanation:*
All scores are included: `(80 + 90 + 100) / 3 = 90`.

`averageScore(true, 80, 90, 100)` returns `95`
*Explanation:*
The lowest score, `80`, is dropped: `(90 + 100) / 2 = 95`.

`averageScore(true, 70)` returns `0`
*Explanation:*
Dropping the only score leaves no scores to average.

**Test Cases**
`averageScore(false)` = 0
`averageScore(false, 80, 90, 100)` = 90
`averageScore(true, 80, 90, 100)` = 95
`averageScore(true, 70)` = 0
`averageScore(false, 10, 20, 30, 40)` = 25
`averageScore(true, 10, 20, 30, 40)` = 30
`averageScore(true, 50, 50, 100)` = 75
