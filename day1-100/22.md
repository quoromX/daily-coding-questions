**Daily Coding Challenge - Day 22: Greedy Text Wrapping**

This challenge is inspired by ascii-art-justify.

Write a function that wraps words into lines of fixed width.

The function `wrapText` should accept:
- `words []string`
- `maxWidth int`

Return a slice of strings where:
- Words are packed greedily (as many as possible per line).
- Keep exactly one space between words on the same line.
- Each line must be padded with trailing spaces so its length is exactly `maxWidth`.
- No word will be longer than `maxWidth`.

This is **left alignment only** (no full justification / no uneven gap distribution).

**Examples**
`wrapText([]string{"This", "is", "an", "example", "text"}, 10)` returns:
`[]string{"This is an", "example   ", "text      "}`

`wrapText([]string{"a", "b", "c", "d"}, 3)` returns:
`[]string{"a b", "c d"}`

**Test Cases**
`wrapText([]string{"a"}, 1)` = `[]string{"a"}`
`wrapText([]string{"ab", "cd", "ef"}, 5)` = `[]string{"ab cd", "ef   "}`
`wrapText([]string{}, 5)` = `[]string{}`
All returned lines must have length `maxWidth`

```go
func wrapText(words []string, maxWidth int) []string {
}
```
