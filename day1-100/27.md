**Daily Coding Challenge - Day 27: Command Line Tokenizer**

CLI tools (like ascii-art projects) need robust argument tokenization.

Write a function that splits a command string into tokens.

The function `tokenizeCommand` should support:
- Spaces as separators
- Double-quoted sections (`"..."`) to keep spaces inside one token
- Escaped quote inside quotes (`\"`)
- Escaped backslash inside quotes (`\\`)

Rules:
- Ignore leading/trailing/multiple spaces outside quotes.
- Return error for unclosed quotes.
- Return tokens in order.

**Examples**
`tokenizeCommand("--align=right \"hello world\" standard")`
returns `[]string{"--align=right", "hello world", "standard"}`

`tokenizeCommand("--color=red \"say \\\"hi\\\"\"")`
returns `[]string{"--color=red", "say \"hi\""}`

`tokenizeCommand("abc  def")` returns `[]string{"abc", "def"}`

**Test Cases**
Empty input returns `[]string{}`
Unclosed quote returns error
Quoted empty string `""` becomes one empty token

func tokenizeCommand(input string) ([]string, error) {
}
