**Daily Coding Challenge - Day 18: Sum Iterator**

Go can use `range` with iterator functions. An iterator function calls `yield` for each value it wants to produce.

Write a function that sums all values from an integer iterator.

The function `sumIterator` should accept one `IntIterator` parameter called `numbers`.

Use `range` to loop over the iterator.

Return the sum of all values produced by the iterator.

If the iterator produces no values, return `0`.

**Examples**
```go
numbers := func(yield func(int) bool) {
	yield(1)
	yield(2)
	yield(3)
}
```

`sumIterator(numbers)` returns `6`
*Explanation:*
The iterator produces `1`, `2`, and `3`.

```go
numbers := func(yield func(int) bool) {
}
```

`sumIterator(numbers)` returns `0`
*Explanation:*
The iterator does not produce any values.

**Test Cases**
```go

```

```

}

```

}
