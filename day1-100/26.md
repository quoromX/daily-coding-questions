**Daily Coding Challenge - Day 26: Rotate a Character Grid Clockwise**

Rotating ASCII art is a common need when transforming sprites or rendering layouts.

Write a function that rotates a 2D rune grid 90 degrees clockwise.

The function `rotateGrid` should accept `grid [][]rune`.

Return a new 2D slice of runes representing the grid after rotation.

Rules:
- The first row of the input becomes the last column of the output.
- Works for square and rectangular grids.
- If grid is empty, return an empty slice.

**Examples**
`rotateGrid([][]rune{{'a','b','c'},{'d','e','f'}})` returns `[][]rune{{'d','a'},{'e','b'},{'f','c'}}`

`rotateGrid([][]rune{{'x'}})` returns `[][]rune{{'x'}}`
