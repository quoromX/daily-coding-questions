**Daily Coding Challenge - Day 6: Minimum Rooms Required**

Given an array of time intervals `(start, end)` for classroom lectures, return the minimum number of rooms required.

Lectures may overlap. If one lecture ends at the same time another begins, they can use the same room.

**Examples**
`minRooms([][2]int{{30, 75}, {0, 50}, {60, 150}})` returns `2`
*Explanation:*
The lecture from `30` to `75` overlaps with both `0` to `50` and `60` to `150`, so two rooms are needed.

`minRooms([][2]int{{0, 10}, {10, 20}})` returns `1`
*Explanation:*
The first lecture ends exactly when the second begins, so they can share one room.

`minRooms([][2]int{{5, 10}, {15, 20}, {25, 30}})` returns `1`
*Explanation:*
None of the lectures overlap.

`minRooms([][2]int{{30, 75}, {0, 50}, {60, 150}})` = 2
`minRooms([][2]int{{0, 30}, {5, 10}, {15, 20}})` = 2
`minRooms([][2]int{{1, 4}, {2, 5}, {3, 6}})` = 3
`minRooms([][2]int{{1, 2}, {2, 3}, {3, 4}})` = 1
