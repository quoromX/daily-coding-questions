**Daily Coding Challenge - Day 4: First Non-Repeating Digit**

Given an integer `n`, return the first digit that does not repeat in the number.
If all digits repeat, return `-1`.

**Examples**
`firstUniqueDigit(-4674)` returns `6`
*Explanation:*
Digits: 4, 6, 7, 4
4 repeats, 6 does not so return 6


`firstUniqueDigit(1212)` returns `-1`
*Explanation:*
All digits repeat so return -1


`firstUniqueDigit(908070)` returns `9`
*Explanation:*
Digits: 9,0,8,0,7,0
9 appears once so return 9


**Test Cases**
`firstUniqueDigit(0)` = 0
`firstUniqueDigit(-49)` = 4
`firstUniqueDigit(1000)` = 1
`firstUniqueDigit(112233)` = -1
`firstUniqueDigit(123451)` = 2
