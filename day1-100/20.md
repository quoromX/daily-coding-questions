**Daily Coding Challenge - Day 20: Valid Brackets**

Stacks help you match opening and closing brackets in the right order.

Write a function that checks whether a string of brackets is valid.

The function `isValidBrackets` should accept a string called `text`.

The string may contain only the characters `()[]{}`.

Return `true` if every opening bracket is closed by the same type of bracket in the correct order. Otherwise return `false`.

If the string is empty, return `true`.

**Examples**
`isValidBrackets("()[]{}")` returns `true`
*Explanation:*
Every opening bracket is closed in the correct order.

`isValidBrackets("([)]")` returns `false`
*Explanation:*
The brackets are closed in the wrong order.

`isValidBrackets("{[]}")` returns `true`
*Explanation:*
The nesting is valid.

**Test Cases**
`isValidBrackets("")` = `true`
`isValidBrackets("()[]{}")` = `true`
`isValidBrackets("([)]")` = `false`
`isValidBrackets("{[]}")` = `true`
`isValidBrackets("(")` = `false`
`isValidBrackets("]")` = `false`
