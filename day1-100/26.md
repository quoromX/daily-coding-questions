**Daily Coding Challenge - Day 26: Shortest Path in a Pixel Grid**

Think of an ASCII canvas where some cells are blocked.

Write a function that finds the shortest path length from `S` to `E` in a grid.

The function `shortestPath` should accept `grid []string` where:
- `'S'` = start
- `'E'` = end
- `'.'` = free cell
- `'#'` = blocked cell

You may move up, down, left, or right.

Return the minimum number of moves from `S` to `E`, or `-1` if unreachable.

Use BFS.

**Examples**
For:
`[]string{"S..", ".#.", "..E"}` returns `4`

For:
`[]string{"S#.", "##.", "..E"}` returns `-1`

`S` equals `E` position not possible in input; assume both exist exactly once
Unreachable layouts return `-1`
