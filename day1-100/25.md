**Daily Coding Challenge - Day 25: Spiral Traverse a Character Grid**

ASCII art is grid-based. Practice traversal with a classic matrix problem.

Write a function that returns the elements of a 2D rune grid in spiral order.

The function `spiralOrder` should accept `grid [][]rune`.

Return a slice of runes visited in clockwise spiral order.

Rules:
- Start from top-left corner.
- Traverse right, down, left, up repeatedly while shrinking bounds.
- If grid is empty, return empty slice.

**Examples**
For grid:
```
a b c
d e f
g h i
```
`spiralOrder` returns `[]rune{'a','b','c','f','i','h','g','d','e'}`

For grid:
```
a b c d
```
returns `[]rune{'a','b','c','d'}`

**Test Cases**
`spiralOrder([][]rune{})` = `[]rune{}`
`spiralOrder([][]rune{{'x'}})` = `[]rune{'x'}`
`spiralOrder` works for rectangular grids (not only squares)

```go
func spiralOrder(grid [][]rune) []rune {
}
```
