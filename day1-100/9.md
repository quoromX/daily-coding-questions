**Daily Coding Challenge - Day 9: Nested Array Sum**

Recursion is when a function calls itself. A recursive function needs a base case so it knows when to stop.

Write a function that calculates the sum of all integers inside a nested array.

The function `nestedSum` should accept one parameter, `items`, which can contain integers or more nested arrays.

If an item is an integer, add it to the total.

If an item is another array, use recursion to calculate the sum of that array, then add it to the total.

If the array is empty, return `0`.

**Examples**
`nestedSum([]any{1, 2, 3})` returns `6`
*Explanation:*
All values are at the top level: `1 + 2 + 3 = 6`.

`nestedSum([]any{1, []any{2, 3}, 4})` returns `10`
*Explanation:*
The nested array contains `2` and `3`, so the total is `1 + 2 + 3 + 4 = 10`.

`nestedSum([]any{[]any{1, []any{2}}, 3})` returns `6`
*Explanation:*
The function must keep going into nested arrays until it reaches the integers.

**Test Cases**
`nestedSum([]any{})` = 0
`nestedSum([]any{1, 2, 3})` = 6
`nestedSum([]any{1, []any{2, 3}, 4})` = 10
`nestedSum([]any{[]any{1, []any{2}}, 3})` = 6
`nestedSum([]any{10, []any{-2, []any{5}}, 1})` = 14
`nestedSum([]any{[]any{}, []any{1, []any{}}, 2})` = 3
