**Daily Coding Challenge - Day 23: Parse Run Arguments**

This question is inspired by the args parsing of ascii-art.

The function should parse `args` (the values after the program name).

Supported flags:
- `--host=<value>` (optional, default: `"localhost"`)
- `--port=<value>` (required, integer `1..65535`)
- `--verbose` (optional boolean switch)
- `--retries=<value>` (optional, default: `3`, integer `0..10`)

Rules:
- Unknown flags return an error.
- Invalid numbers return an error.
- Missing required `--port` returns an error.
- If a flag appears multiple times, use the last value.

**Examples**
`parseRunArgs([]string{"--port=8080"})`
returns `Config{Host: "localhost", Port: 8080, Verbose: false, Retries: 3}, nil`

`parseRunArgs([]string{"--host=0.0.0.0", "--port=9000", "--verbose", "--retries=5"})`
returns `Config{Host: "0.0.0.0", Port: 9000, Verbose: true, Retries: 5}, nil`

`parseRunArgs([]string{"--verbose"})`
returns `error` (missing `--port`)

**Test Cases**

`parseRunArgs([]string{"--port=70000"})` = `error`
`parseRunArgs([]string{"--retries=11", "--port=8080"})` = `error`

`parseRunArgs([]string{"--verbose"})` = `error`
`parseRunArgs([]string{"--unknown=1", "--port=8080"})` = `error`
