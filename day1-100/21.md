**Daily Coding Challenge - Day 21: Group Anagrams**

Hash maps can help group values that share the same signature.

Write a function that groups words that are anagrams of each other.

The function `groupAnagrams` should accept one parameter, `words`, which is a slice of strings.

Two words are anagrams if they contain the same letters with the same counts, in any order.

Return a slice of groups, where each group is a slice of strings that are anagrams.

The order of groups does not matter.

Inside each group, keep words in the same order they appeared in the input.

If the input is empty, return an empty slice.

Assume all words are lowercase English letters.

**Examples**
`groupAnagrams([]string{"eat", "tea", "tan", "ate", "nat", "bat"})` returns `[][]string{{"eat", "tea", "ate"}, {"tan", "nat"}, {"bat"}}`
*Explanation:*
Words with the same letter-count signature are placed in the same group.

`groupAnagrams([]string{""})` returns `[][]string{{""}}`
*Explanation:*
An empty string is still a valid word and forms one group.

`groupAnagrams([]string{"abc", "def"})` returns `[][]string{{"abc"}, {"def"}}`
*Explanation:*
No two words are anagrams, so each word is its own group.
