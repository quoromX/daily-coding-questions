**Daily Coding Challenge - Single Digit**

Given a positive integer named `n`, add all the digits in the number until it becomes a single digit and return it.

**Example 1:**
`singleDigit(467) -> 8`
Explanation:
467 = 4 + 6 + 7 = 17
17 = 1 + 7 = 8
8 is a single digit, return it

**Example 2:**
`singleDigit(21) -> 3`
21 = 2 + 1 = 3
3 is a single digit, return it


**Test**
singleDigit(49) should return 4.
singleDigit(1000) should return 1.
singleDigit(6) should return 6.
singleDigit(19564) should return 7.
