**Daily Coding Challenge - Day 3: Fraction**

Given two integers `n` and `d` which are numerators and denominators respectively, return fraction represention of the two digits in the format `<WHOLE-NUMBER> <NUMERATOR>/<DENOMINATOR>`.
If `d` is `0`, return `"-1"`.

**Examples**
`getFraction(3, 4)` returns `"3/4"`

`getFraction(7, 4)` returns `"1 3/4"`
*Explanation:*
7 / 3 = 1 remainder 3 so `"1 3/4"`

`getFraction(2, 4)` returns `"2/4"`

`getFraction(0, 4)` returns `"0"`

`getFraction(10, 2)` returns `"5"`
*Explanation:*
The remainder of dividing 10 by 2 is 0 so `"5 0/2"` is zero 

`getFraction(4, 3)` returns `"-1"`
*Explanation.*
Division by zero is not possible in standard arithemetic
