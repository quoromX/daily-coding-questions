**Daily Coding Challenge - Day 23: Merge Highlight Ranges**

Coloring substrings often creates overlapping highlight intervals.

Write a function that merges overlapping or touching ranges.

The function `mergeRanges` should accept `ranges`, where each range is `[start, end)` (end is exclusive).

Return merged ranges sorted by start index.

Rules:
- Two ranges should merge if they overlap or touch (`next.start <= current.end`).
- Ignore invalid ranges where `start >= end`.
- If input is empty, return empty output.

**Examples**
`mergeRanges([][2]int{{1,4},{2,6},{8,10},{10,12}})` returns `[][2]int{{1,6},{8,12}}`

`mergeRanges([][2]int{{5,7},{1,3}})` returns `[][2]int{{1,3},{5,7}}`

`mergeRanges([][2]int{{3,3},{2,2}})` returns `[][2]int{}`

**Test Cases**
`mergeRanges([][2]int{})` = `[][2]int{}`
`mergeRanges([][2]int{{0,1}})` = `[][2]int{{0,1}}`
`mergeRanges([][2]int{{1,5},{5,9}})` = `[][2]int{{1,9}}`
`mergeRanges([][2]int{{1,4},{6,8}})` = `[][2]int{{1,4},{6,8}}`
