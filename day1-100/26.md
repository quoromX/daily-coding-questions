**Daily Coding Challenge - Day 26: Run-Length Encode a String**

Run-length encoding is a simple compression technique often used for ASCII art and sprite data.

Write a function that compresses a string using run-length encoding.

The function `runLengthEncode` should accept a string called `text`.

Return a string where each run of consecutive identical characters is replaced by the character followed by its count.

Rules:
- A run of length 1 should still include the count (e.g. `"1"`, not omitted).
- If the string is empty, return an empty string.
- Counts can be more than one digit.

**Examples**
`runLengthEncode("aaabbc")` returns `"a3b2c1"`

`runLengthEncode("abcd")` returns `"a1b1c1d1"`

**Test Cases**
`runLengthEncode("")` = `""`

`runLengthEncode("abcd")` = `"a1b1c1d1"`
`runLengthEncode("aaaaaaaaaaaa")` = `"a12"`
`runLengthEncode("zzzzzzzzzzz")` = `"z11"`
