**Daily Coding Challenge - Day 19: Move Zeros**

Slices let you work with a sequence of values and update the values by index.

Write a function that moves all zeros in an integer slice to the end.

The function `moveZeros` should accept a slice of integers called `nums`.

Move every `0` value to the end of the slice.

Keep the non-zero values in the same order they appeared in the input.

Modify the provided slice in place and return it.

If the slice is empty, return an empty slice.

Negative numbers are non-zero values and should keep their order.

**Examples**
`moveZeros([]int{0, 1, 0, 3, 12})` returns `[]int{1, 3, 12, 0, 0}`
*Explanation:*
The non-zero values are `1`, `3`, and `12`, so they move to the front in the same order.

`moveZeros([]int{4, 0, -2, 0, 5})` returns `[]int{4, -2, 5, 0, 0}`
*Explanation:*
`-2` is not zero, so it stays with the other non-zero values.

`moveZeros([]int{1, 2, 3})` returns `[]int{1, 2, 3}`
*Explanation:*
There are no zeros to move.

**Test Cases**

`moveZeros([]int{0})` = `[]int{0}`

`moveZeros([]int{4, 0, -2, 0, 5})` = `[]int{4, -2, 5, 0, 0}`
`moveZeros([]int{1, 2, 3})` = `[]int{1, 2, 3}`
`moveZeros([]int{0, 0, 7})` = `[]int{7, 0, 0}`
